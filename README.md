# Coffee Shop Sales Analysis Dashboard
### Project Overview

This project analyzes coffee shop sales data to understand customer behavior, sales trends, and product performance. An interactive dashboard was created to provide clear business insights that support data-driven decision-making.

### Key Metrics

* Total Sales: $698,812

* Total Footfall: 149,116

* Average Bill per Person: $4.69

* Average Orders per Person: 1.44

### Key Insights

* Morning hours (8 AM – 10 AM) generate the highest sales volume.

* Coffee and tea are the primary revenue-driving categories.

* Espresso and specialty beverages are the top-performing products.

* Weekdays, especially Friday, outperform weekends in order volume.

* Hell’s Kitchen is the highest-performing store location.

### Tools Used

* Microsoft Excel (Data Cleaning, Analysis, Dashboard Creation)

### Outcome

The dashboard delivers a concise view of sales performance, customer trends, and operational patterns, enabling stakeholders to identify growth opportunities and optimize business strategies.
